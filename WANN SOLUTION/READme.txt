WANN SOLUTION WEBSITE 
==========================

Description:
This is a responsive business website for WANN SOLUTION, a fictional technology consulting company. The website includes four main pages: Home, About Us, Services, and Contact.

Features Implemented:
1. HTML Structure:
   - Proper HTML5 semantic elements (header, nav, section, article, aside, footer)
   - 4 interlinked pages
   - Lists, tables, and appropriate headings
   - Images with alt text

2. CSS Styling:
   - External CSS file
   - Responsive design with media queries
   - CSS Box Model implementation
   - Flexbox and CSS Grid for layout

3. JavaScript Features:
   - Dark/Light mode toggle (saves preference to localStorage)
   - Form validation for contact page
   - XML data loading for testimonials

4. XML Integration:
   - data.xml file stores testimonial data
   - JavaScript loads and displays this data on the homepage

5. Accessibility:
   - ARIA attributes for better screen reader support
   - Proper heading hierarchy
   - Sufficient color contrast
   - Keyboard navigable

Files Included:
- index.html       - Home page
- about.html      - About Us page
- services.html   - Services page
- contact.html    - Contact page
- styles.css      - CSS stylesheet
- script.js       - JavaScript file
- data.xml        - XML data file
- README.txt      - This file

Note: Image files referenced in the code are not included in this submission. In a production environment, you would need to create or source appropriate images and place them in an images/ directory.
